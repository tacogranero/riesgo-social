Índice de Riesgo Social (IRS)

Monitoreo automático del riesgo económico-social argentino

¿Qué es el IRS?

El Índice de Riesgo Social mide el nivel de presión económica sobre las familias argentinas en base a datos oficiales. Incluye variables como el ingreso modal, canasta básica total y alimentaria, inflación, tarifas, transporte, medicamentos y desempleo.

Datos Automáticos

Este sistema utiliza datos en tiempo real desde APIs públicas como las del BCRA, INDEC y otras fuentes para calcular el IRS. El índice se actualiza periódicamente y ofrece alertas e indicadores clave para formular políticas públicas.

Metodología

IRS = (Gastos Totales / Ingreso Modal) * 100 + Inflación + Desempleo

Cuando supera el 80%, se recomienda intervenir con políticas progresistas: subsidios, fortalecimiento del empleo, y mejora del poder adquisitivo.

Visualización

La aplicación incluye gráficos dinámicos y exportación de resultados.
